# Age Calculator

A simple and clean age calculator web app built by AJ STUDIOZ. Enter your birthdate to instantly see your exact age and how many days until your next birthday.

## Features

- Calculates exact age in years, months, and days
- Shows days remaining until next birthday
- Validates future dates with error handling
- Clean, responsive UI centered on all screen sizes

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

No build tools or dependencies required. Just open `index.html` in any browser:

```bash
# Clone the repo
git clone https://github.com/kamesh14151/KAMESHAJ.git

# Open in browser
open index.html
```
